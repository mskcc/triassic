# triassic

Containers to faciliate the running of old (pre-disaster) JUNO scripts

## Program/Application List:

Unless otherwise specified use latest convenient version

- Samtools 
- htslib
- Bedtools
- R: version >= 4.2.x (can _not_ be 3.x)
- Python3: version >= 3.7
